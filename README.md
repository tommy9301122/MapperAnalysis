# osu! Mapper Analysis
 osu! mapper analysis  
 
### Requirements
`
pip install osuapi pandas numpy request pyecharts
`

### Analysis Info
* **Achievement**:  
Mapping Age (first map submit date ~ last map update date)  
Map set amount  
Rank map set amount  
Favourite count  

* **Map set**:  
First map and submit date  
First rank map and approved date  
Efforts map and spend time  

* **Style**:  
Mode  
Language  
Genre  
Length  
Star rating  
BPM(X)
