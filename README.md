# osu! Mapper Analysis
 osu! mapper analysis  
 
### 環境需求
`
pip install osuapi pandas numpy request
`

### 項目
* **麻婆資訊**:  
1. 做圖年齡(第一次上傳圖~最後一次上傳圖)  
2. 做圖數量(map set)  
3. rank數量(map set)及總遊玩次數  
4. 總收藏數量  

* **麻婆做圖史**:  
1. 第一張圖及第一次上傳時間  
2. 第一張Rank圖及上榜時間  
3. 開始作圖後花了多久Rank了第一張圖  
4. 花最多時間mapping的圖及花費的時間  

* **麻婆風格統計**:  
1. 模式統計  
2. 歌曲語言統計  
3. 歌曲類型統計  
4. 圖長統計  
5. 難度統計  
6. 做圖平均BPM
